## What to Include

The more context you provide, the faster we can act on it.

### For a bug report
- What you were trying to do
- What happened vs. what you expected
- Steps to reproduce the issue
- Your platform, device, and app or browser version
- Screenshots, recordings, or error messages (if available)

### For a feature request
- The problem or need you're trying to solve
- What you'd like Attryx to do
- Any context on how you'd use it

## Important: Sensitive Issues

This is a public repository. **Do not post** billing details, account credentials, personal information, or any other sensitive data in an issue.

For anything involving your account, billing, or a matter you'd prefer to keep private, please email us directly:

**[support@attryx.ai](mailto:support@attryx.ai)**

We aim to respond to all support emails within a few business days.

---

## Tips for Faster Review

- Use a clear, specific title — it helps us review and prioritize faster.
- One issue per report keeps things focused and easy to track.
- Check existing issues first to avoid duplicates.
- If you're unsure whether something is a bug or expected behavior, open an issue anyway — we'd rather hear about it.
